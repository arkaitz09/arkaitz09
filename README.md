## Featured project — Aries

> A web app for a healthcare team that processes Excel files and generates WhatsApp messages for families with appointment information.

**Tech:** Spring Boot · JWT (HttpOnly cookies) · React + Vite · Railway · Cloudflare Pages + Workers

**What makes it interesting:**
- Deployed and used by **3 real users** for 3+ months
- Custom reverse proxy via Cloudflare Worker (`/api/*` → Railway backend) — zero CORS issues, one clean domain
- Auth with HttpOnly cookies (not localStorage) — intentional security decision
